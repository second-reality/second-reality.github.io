GestionClients
##############

Afin de pouvoir accéder à ce fichier, vous devez accepter d'exécuter les macros qu'il contient.
Télécharger GestionClients.xls_

.. _GestionClients.xls: /_static/projets/gestionclients/GestionClientsV1.2.xls

Les instructions de configuration / utilisation sont contenues dans le fichier Manuel.doc_

.. _Manuel.doc: /_static/projets/gestionclients/Manuel.doc

Enjoy ;)

Fichier créé pour Xp et Excel 2003 / compatible Vista et 2007
Optimisé pour thème classique Vista - Xp
