GestionClients¶

Afin de pouvoir accéder à ce fichier, vous devez accepter d’exécuter les macros qu’il contient. Télécharger GestionClients.xls

Les instructions de configuration / utilisation sont contenues dans le fichier Manuel.doc

Enjoy ;)

Fichier créé pour Xp et Excel 2003 / compatible Vista et 2007 Optimisé pour thème classique Vista - Xp
